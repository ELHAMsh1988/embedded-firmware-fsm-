# Embedded FSM Project

Dieses Projekt enthält eine Embedded-Firmware für Mikrocontroller mit Finite-State-Machines (FSM). 
Es demonstriert die Implementierung, Simulation und Debugging von Steuerungslogik.
# Embedded Firmware – FSM Example

## Overview
This project demonstrates a simple embedded firmware architecture using
Finite State Machines (FSM) written in Embedded C.

## Features
- Finite State Machine based control logic
- Modular firmware structure
- Interrupt-ready design concept
- Hardware-near embedded programming

## Project Structure
- `src/` : Embedded C source code
- `docs/` : FSM diagram, simulation output and documentation

## Simulation / Output
This project was validated using simulation and logical verification.
Example outputs and FSM diagrams are available in the docs folder.

## Technologies
- Embedded C
- Microcontroller firmware concepts
- FSM design
- Git version control
